或多或少优化了其中一些通用部分的逻辑，虽然这个过程是那么的艰难，但还是强忍着痛苦完成了这些工作，让一些不那么完善的代码得到一定的优化。在以后的定制中，或许能减轻一些工作，这也许就足够了。

也刚好借此机会，总结整理一些自己对于代码重构优化[1]的想法。本文总结整理了xfusion[2]条件查询优化的一些粗浅经验之谈，如果有不准确的地方，还望指点。

条件封装逻辑 ​

洋洋洒洒200+代码，通过“淳朴”的new对象的方式构造每一个条件，简单粗暴。

虽然没有办法说目前代码逻辑是错的，但是这种堆叠代码的方式，无论是扩展性还是健壮性好像都差点意思。

问题分析 ​

其实根据xfusion查询的请求结构体，不难发现其中可以进行归纳总结出的统一的检索条件策略，而我们是不是其实只需要实现了这些检索策略，那么条件的扩展也就会变得相对简单了。

总体设计思路 ​

总体设计 ​

基本设计思路 ​

  • 将条件进行统一封装成通用的“策略”[3]；
  • 通过注解的方式描述属性条件特征；
  • 复杂条件通过“处理器”实现；
  • 通过“装饰类”[4]对查询接口进行包装；

策略模式 ​

对于xfusion本身支持的单条件：in、like、range、terms、timerange，通过定义通用的条件策略来实现单条件场景。

处理器模式 ​

因为部分条件无法通过单一的策略实现条件组装，比如plateNo（车牌条件，需要考虑精确查询，模糊查询以及无车牌的情况），axlesNumber（轴数条件，其中‘六轴以上’需要通过区间条件实现）。

所以就想到通过处理器的方式来处理这些复杂条件，即保证了对于复杂条件组装的需求，也保留了一定的扩展性。

装饰者模式 ​

通过装饰xfusion的IVehicleSearchServiceFeign Client服务，将原本抽象的QueryFieldOptions查询条件和LinkedHashMap返回结构，包装成VehiclePassReqDTO和VehiclePassRespDTOPOJO对象，对于调用方更加友好和通俗易懂。

InFieldOptionStrategy ​

java
/**
 * in多条件查询策略实现
 *
 * @author yuanzp
 * @date 2022/3/26 10:44
 */
public class InFieldOptionStrategy extends AbstractFieldOptionStrategy implements IFieldOptionStrategy<String, List<String>, FieldOption> {
    public InFieldOptionStrategy() {
        super(FieldOption.TYPE_IN);
    }

    public InFieldOptionStrategy(String grammar) {
        super(grammar);
    }

    @Override
    public FieldOption doOperation(String fieldName, List<String> keyWords) {
        if (CollUtil.isEmpty(keyWords)) {
            return null;
        }
        String newKeyWords = StrUtil.join(StrUtil.COMMA, CollUtil.newHashSet(CollUtil.removeBlank(keyWords)));
        return new FieldOption(fieldName, newKeyWords, grammar);
    }
}

LikeFieldOptionStrategy ​

java
/**
 * like条件查询策略实现
 *
 * @author yuanzp
 * @date 2022/3/26 11:03
 */
public class LikeFieldOptionStrategy extends AbstractFieldOptionStrategy implements IFieldOptionStrategy<String, String, FieldOption> {
    public static final String QUESTION_MARK = "?";
    public static final String ASTERISK = "*";

    public LikeFieldOptionStrategy() {
        super(FieldOption.TYPE_LIKE);
    }

    public LikeFieldOptionStrategy(String grammar) {
        super(grammar);
    }

    @Override
    public FieldOption doOperation(String fieldName, String keyWords) {
        if (StrUtil.isBlank(keyWords)) {
            return null;
        }
        keyWords = StrUtil.containsAny(keyWords, ASTERISK, QUESTION_MARK) ? keyWords
                : StrUtil.concat(true, ASTERISK, keyWords, ASTERISK);
        return new FieldOption(fieldName, keyWords, grammar);
    }
}

RangeFieldOptionStrategy ​

java
/**
 * 区间查询策略实现
 *
 * @author yuanzp
 * @date 2022/3/26 10:28
 */
public class RangeFieldOptionStrategy extends AbstractFieldOptionStrategy implements IFieldOptionStrategy<String, Pair<Object, Object>, FieldOption> {

    public RangeFieldOptionStrategy() {
        super(FieldOption.TYPE_RANGE);
    }

    public RangeFieldOptionStrategy(String grammar) {
        super(grammar);
    }

    @Override
    public FieldOption doOperation(String fieldName, Pair<Object, Object> keyWords) {
        //左区间默认0
        BigDecimal leftInterval = Optional.ofNullable(keyWords).map(Pair::getLeft)
                .map(this::toBigDecimal).orElse(BigDecimal.ZERO);
        //右区间默认Integer.MAX_VALUE
        BigDecimal rightInterval = Optional.ofNullable(keyWords).map(Pair::getRight)
                .map(this::toBigDecimal).orElse(BigDecimal.valueOf(Integer.MAX_VALUE));
        String newKeyWords = StrUtil.concat(true, leftInterval.toPlainString(), StrUtil.COMMA, rightInterval.toPlainString());
        return new FieldOption(fieldName, newKeyWords, grammar);
    }

    /**
     * 强制类型转换
     * @param o
     * @return
     */
    private BigDecimal toBigDecimal(Object o) {
        if (o instanceof Integer) {
            return NumberUtil.toBigDecimal((Integer) o);
        }
        if (o instanceof String) {
            return NumberUtil.toBigDecimal((String) o);
        }
        if (o instanceof BigDecimal) {
            return (BigDecimal) o;
        }
        return null;
    }
}

TimeRangeFieldOptionStrategy ​

java
/**
 * 时间区间查询策略实现
 *
 * @author yuanzp
 * @date 2022/3/26 10:14
 */
public class TimeRangeFieldOptionStrategy extends AbstractFieldOptionStrategy implements IFieldOptionStrategy<String, Pair<Date, Date>, FieldOption> {

    public TimeRangeFieldOptionStrategy() {
        super(FieldOption.TYPE_TIMERANGE);
    }

    public TimeRangeFieldOptionStrategy(String grammar) {
        super(grammar);
    }

    @Override
    public FieldOption doOperation(String fieldName, Pair<Date, Date> keyWords) {
        //无起始时间，默认1970-01-01T08:00:00Z
        Date beginTime = Optional.ofNullable(keyWords).map(Pair::getLeft).orElse(DateUtil.date(0));
        //无结束时间，默认当前时间+12个月
        Date endTime = Optional.ofNullable(keyWords).map(Pair::getRight).orElse(DateUtil.offsetMonth(DateUtil.date(), 12));
        String newKeyWords = StrUtil.concat(true, DateUtil.format(beginTime, DatePattern.UTC_PATTERN), StrUtil.COMMA, DateUtil.format(endTime, DatePattern.UTC_PATTERN));
        return new FieldOption(fieldName, newKeyWords, grammar);
    }
}

WhetherFieldOptionStrategy ​

java
/**
 * 是或否多条件查询策略实现
 *
 * @author yuanzp
 * @date 2022/4/8 16:26
 */
public class WhetherFieldOptionStrategy extends AbstractFieldOptionStrategy implements IFieldOptionStrategy<String, String, FieldOption> {
    public static final String ALL = "-1";
    public static final String YES = "1";
    public static final String NO = "0";

    public WhetherFieldOptionStrategy() {
        super(null);
    }

    @Override
    public FieldOption doOperation(String fieldName, String keyWords) {
        if (StrUtil.isBlank(keyWords)) {
            return null;
        }
        if (ALL.equals(keyWords)) {
            String newKeyWords = StrUtil.join(StrUtil.COMMA, CollUtil.newHashSet(YES, NO));
            return new FieldOption(fieldName, newKeyWords, FieldOption.TYPE_IN);
        }
        return new FieldOption(fieldName, keyWords, FieldOption.TYPE_TEARMS);
    }
}

AxlesNumberHandler ​

java
/**
 * 车辆轴数处理拦截器
 *
 * @author yuanzp
 * @date 2022/3/26 17:26
 */
public class AxlesNumberHandler implements FieldMultipleOptionsHandler<VehiclePassReqDTO> {
    private static final Integer AXLE_NUM_OF_SIX = 6;
    private static final String AXLE_NUM_FIELD_NAME = "axleNum";

    private IFieldOptionStrategy<String, List<String>, FieldOption> inFieldOptionStrategy;
    private IFieldOptionStrategy<String, Pair<Object, Object>, FieldOption> rangeFieldOptionStrategy;

    public AxlesNumberHandler(InFieldOptionStrategy inFieldOptionStrategy, RangeFieldOptionStrategy rangeFieldOptionStrategy) {
        this.inFieldOptionStrategy = inFieldOptionStrategy;
        this.rangeFieldOptionStrategy = rangeFieldOptionStrategy;
    }

    @Override
    public void execute(VehiclePassReqDTO vehiclePassReqDTO, List<FieldMultipleOptions> fieldMultipleOptionsList) {
        String axleNum = vehiclePassReqDTO.getAxleNum();
        List<String> axleNumList = vehiclePassReqDTO.getAxleNums();
        if (CollUtil.isEmpty(axleNumList) && StrUtil.isBlank(axleNum)) {
            return;
        }
        //axleNums条件优先级高于axleNum
        axleNumList = CollUtil.isEmpty(axleNumList) ? CollUtil.newArrayList(axleNum) : axleNumList;
        FieldMultipleOptions fieldMultipleOptions = new FieldMultipleOptions();
        fieldMultipleOptionsList.add(fieldMultipleOptions);
        fieldMultipleOptions.getOr().add(inFieldOptionStrategy.doOperation(AXLE_NUM_FIELD_NAME, axleNumList));
        if (axleNumList.contains(String.valueOf(AXLE_NUM_OF_SIX))) {
            //包含六轴以上，增加七轴及以上的复合条件
            fieldMultipleOptions.getOr().add(rangeFieldOptionStrategy.doOperation(AXLE_NUM_FIELD_NAME,
                    Pair.of(AXLE_NUM_OF_SIX, Integer.MAX_VALUE)));
        }
    }
}

PlateNoHandler ​

java
/**
 * 车牌处理拦截器
 *
 * @author yuanzp
 * @date 2022/3/26 17:23
 * @digression done is better than perfect.
 */
public class PlateNoHandler implements FieldMultipleOptionsHandler<VehiclePassReqDTO> {
    private static final String PLATE_NO_FIELD_NAME = "plateNo";

    @Resource
    private IFieldOptionStrategy<String, String, FieldOption> likeFieldOptionStrategy;

    public PlateNoHandler(LikeFieldOptionStrategy likeFieldOptionStrategy) {
        this.likeFieldOptionStrategy = likeFieldOptionStrategy;
    }

    @Override
    public void execute(VehiclePassReqDTO vehiclePassReqDTO, List<FieldMultipleOptions> fieldMultipleOptionsList) {
        String plateState = vehiclePassReqDTO.getPlateState();
        String plateNo = vehiclePassReqDTO.getPlateNo();
        //车牌状态为不限且车牌为空，跳过复合条件组装
        Boolean flag = (StrUtil.isBlank(plateState) || Constants.PLATE_STATE_ALL.equals(plateState))
                && StrUtil.isBlank(plateNo);
        if (flag) {
            return;
        }
        FieldMultipleOptions fieldMultipleOptions = new FieldMultipleOptions();
        fieldMultipleOptionsList.add(fieldMultipleOptions);
        //正常
        if (Constants.PLATE_STATE_NORMAL.equals(plateState)) {
            if (StrUtil.isNotBlank(plateNo)) {
                fieldMultipleOptions.getAnd().add(likeFieldOptionStrategy.doOperation(PLATE_NO_FIELD_NAME, plateNo));
            }
            Constants.NO_PLATE_SET.forEach(o -> {
                fieldMultipleOptions.getNot().add(likeFieldOptionStrategy.doOperation(PLATE_NO_FIELD_NAME, o));
            });
        }
        //未识别
        else if (Constants.PLATE_STATE_UNRECOGNIZED.equals(plateState)) {
            Constants.NO_PLATE_SET.forEach(o -> {
                fieldMultipleOptions.getOr().add(likeFieldOptionStrategy.doOperation(PLATE_NO_FIELD_NAME, o));
            });
        }
        //车牌状态不限
        else {
            if (StrUtil.isNotBlank(plateNo)) {
                fieldMultipleOptions.getAnd().add(likeFieldOptionStrategy.doOperation(PLATE_NO_FIELD_NAME, plateNo));
            }
        }
    }
}

/**
 * 过车请求参数策略处理器
 * （通过条件策略（单条件）+处理器（复合条件）的方式实现）<br/>
 * <p>条件策略接口详见 {@link IFieldOptionStrategy}，内置如下：</p>
 * <ol>
 *      <li>{@link InFieldOptionStrategy}：多条件策略 </li>
 *      <li>{@link LikeFieldOptionStrategy}：模糊条件策略 </li>
 *      <li>{@link RangeFieldOptionStrategy}：区间条件策略 </li>
 *      <li>{@link TimeRangeFieldOptionStrategy}：时间区间条件策略 </li>
 *      <li>{@link TermsFieldOptionStrategy}：精确检索条件策略 </li>
 *      <li>{@link WhetherFieldOptionStrategy}：自定义是否条件策略 </li>
 * </ol>
 *
 * 处理器接口详见 {@link IQueryFieldOptionsHandler}，事例如下：</p>
 * <ol>
 *      <li>{@link AxlesNumberHandler}：车辆轴数处理器</li>
 *      <li>{@link PlateNoHandler}：车牌条件处理器</li>
 * </ol>
 *
 * <p>策略及处理器通过注解的方式与对象和属性进行绑定，详细参考{@link VehiclePassReqDTO}~</p>
 *
 * @author yuanzp
 * @date 2022/3/25 15:44
 * @digression done is better than perfect.
 */
@Log4j
public class VehiclePassQueryFieldOptionsContext {
    private final Map<Class<? extends IFieldOptionStrategy>, IFieldOptionStrategy> DEFAULT_FIELD_OPTION_STRATEGYS = new HashMap<>();
    private final Map<Class<? extends ISortStrategy>, ISortStrategy> DEFAULT_SORT_STRATEGYS = new HashMap<>();
    private final Map<Class<? extends IQueryFieldOptionsHandler>, IQueryFieldOptionsHandler> DEFAULT_QUERY_FIELD_OPTIONS_HANDLERS = new HashMap<>();

    public VehiclePassQueryFieldOptionsContext() {
    }

    public VehiclePassQueryFieldOptionsContext(List<IFieldOptionStrategy> fieldOptionStrategys,
                                               List<ISortStrategy> sortStrategies,
                                               List<IQueryFieldOptionsHandler> queryFieldOptionsHandlers) {
        fieldOptionStrategys.forEach(this::addIFieldOptionStrategy);
        sortStrategies.forEach(this::addISortStrategy);
        queryFieldOptionsHandlers.forEach(this::addIQueryFieldOptionsHandler);
    }


    /**
     * 执行策略
     * @param vehiclePassReqDTO
     * @return
     */
    public <R extends VehiclePassReqDTO> QueryFieldOptions executeStrategy(R vehiclePassReqDTO) {
        QueryFieldOptions queryFieldOptions = newDefault();
        queryFieldOptions.setSearchResultPosition(Optional.ofNullable(vehiclePassReqDTO.getCurrentPage()).orElse(Constants.DEFAULT_RESOURCE_PAGE_NO));
        queryFieldOptions.setMaxResults(Optional.ofNullable(vehiclePassReqDTO.getPageSize()).orElse(Constants.DEFAULT_PAGE_SIZE));
        queryFieldOptions.setFields(Optional.ofNullable(vehiclePassReqDTO.getFields()).orElse("*"));
        Field[] fields = ReflectUtil.getFields(vehiclePassReqDTO.getClass());
        for (Field field : fields) {
            fieldOptionBulider(queryFieldOptions.getFieldOptions(), field, vehiclePassReqDTO);
            sortBulider(queryFieldOptions.getSort(), field, vehiclePassReqDTO);
        }
        Class<? extends VehiclePassReqDTO> vehiclePassReqDTOClass = vehiclePassReqDTO.getClass();
        FieldMultipleOptionStrategy annotation = vehiclePassReqDTOClass.getAnnotation(FieldMultipleOptionStrategy.class);
        //AnnotationUtils.getAnnotation(vehiclePassReqDTOClass, FieldMultipleOptionStrategy.class);
        Class<? extends FieldMultipleOptionsHandler>[] fieldMultipleOptionsHandlers = annotation.value();
        if (fieldMultipleOptionsHandlers.length > 0) {
            for (Class<? extends FieldMultipleOptionsHandler> fieldMultipleOptionsHandler : fieldMultipleOptionsHandlers) {
                IQueryFieldOptionsHandler queryFieldOptionsHandler = DEFAULT_QUERY_FIELD_OPTIONS_HANDLERS.get(fieldMultipleOptionsHandler);
                queryFieldOptionsHandler.execute(vehiclePassReqDTO, queryFieldOptions.getFieldMultipleOptions());
            }
        }
        return queryFieldOptions;
    }

    /**
     * 单条件策略构造器
     * @param fieldOptionsList
     * @param field
     * @param vehiclePassReqDTO
     */
    private <R extends VehiclePassReqDTO> void fieldOptionBulider(List<FieldOption> fieldOptionsList, Field field, R vehiclePassReqDTO) {
        FieldOptionStrategy fieldOptionStrategy = field.getAnnotation(FieldOptionStrategy.class);
        if (ObjectUtil.isNull(fieldOptionStrategy)) {
            return;
        }
        String fieldName_ = fieldOptionStrategy.fieldName();
        FieldOptionType fieldOptionType = fieldOptionStrategy.fieldOptionType();
        String fieldName = field.getName();
        Object fieldValue = ReflectUtil.getFieldValue(vehiclePassReqDTO, field);
        FieldOption fieldOption = null;
        Class<? extends IFieldOptionStrategy> fieldOptionStrategyClass;
        if (FieldOptionType.CUSTOM.equals(fieldOptionType)) {
            fieldOptionStrategyClass = fieldOptionStrategy.fieldOptionStrategy();
        } else {
            fieldOptionStrategyClass = fieldOptionType.getFieldOptionStrategyClass();
        }
        IFieldOptionStrategy iFieldOptionStrategy = DEFAULT_FIELD_OPTION_STRATEGYS.get(fieldOptionStrategyClass);
        if (ObjectUtil.isNotNull(iFieldOptionStrategy) && ObjectUtil.isNotNull(fieldValue)) {
            fieldName_ = StrUtil.isBlank(fieldName_) ? fieldName : fieldName_;
            fieldOption = (FieldOption) iFieldOptionStrategy.doOperation(fieldName_, fieldValue);
        }
        if (ObjectUtil.isNotNull(fieldOption)) {
            fieldOptionsList.add(fieldOption);
        }
    }

    /**
     * 排序规则策略构造器
     * @param sortList
     * @param field
     * @param vehiclePassReqDTO
     */
    private <R extends VehiclePassReqDTO> void sortBulider(List<Sort> sortList, Field field, R vehiclePassReqDTO) {
        SortStrategy sortStrategy = field.getAnnotation(SortStrategy.class);
        if (ObjectUtil.isNull(sortStrategy)) {
            return;
        }
        String fieldName = sortStrategy.fieldName();
        Object fieldValue = ReflectUtil.getFieldValue(vehiclePassReqDTO, field);
        SortType sortType = sortStrategy.defaultSortType();
        String keyword = sortType.getKeyword();
        Class<? extends ISortStrategy> sortStrategyClass = sortType.getSortStrategyClass();
        ISortStrategy iSortStrategy = DEFAULT_SORT_STRATEGYS.get(sortStrategyClass);
        Sort sort = (Sort) iSortStrategy.doOperation(fieldName, ObjectUtil.isNotNull(fieldValue) ? fieldValue : keyword);
        sortList.add(sort);
    }

    /**
     * 检索对象初始化构造器
     * @return
     */
    private QueryFieldOptions newDefault() {
        QueryFieldOptions queryFieldOptions = new QueryFieldOptions();
        queryFieldOptions.setType(Constants.XFUSION_TYPE_VEHICLEPASS);
        return queryFieldOptions;
    }

    private <T> Class<T> getClass(T t) {
        return (Class<T>) t.getClass();
    }

    /**
     * 载入条件策略
     * @param iFieldOptionStrategy
     */
    public void addIFieldOptionStrategy(IFieldOptionStrategy iFieldOptionStrategy) {
        DEFAULT_FIELD_OPTION_STRATEGYS.put(getClass(iFieldOptionStrategy), iFieldOptionStrategy);
    }

    /**
     * 载入排序策略
     * @param iSortStrategy
     */
    public void addISortStrategy(ISortStrategy iSortStrategy) {
        DEFAULT_SORT_STRATEGYS.put(getClass(iSortStrategy), iSortStrategy);
    }

    /**
     * 载入复合条件拦截器
     * @param iQueryFieldOptionsHandler
     */
    public void addIQueryFieldOptionsHandler(IQueryFieldOptionsHandler iQueryFieldOptionsHandler) {
        DEFAULT_QUERY_FIELD_OPTIONS_HANDLERS.put(getClass(iQueryFieldOptionsHandler), iQueryFieldOptionsHandler);
    }

}

/**
 * 过车查询装饰类
 * <p>
 *     基线默认查询条件对象为{@link VehiclePassReqDTO}，属性和xfusion查询条件相对应；<br/>
 *     返回对象为{@link VehiclePassRespDTO}。<br/>
 *     如果在后期定制中需要进行属性扩展（即xfusion进行字段扩充），建议添加到{@link ExtVehiclePassReqDTO}对象，<br/>
 *     然后通过自定义检索方法{@link #customPaging(VehiclePassReqDTO, Class)}进行查询，方便基线时候进行回归。
 * </p>
 *
 * @author yuanzp
 * @date 2022/3/28 14:45
 * @digression done is better than perfect.
 */
@Slf4j
@Component
public class VehiclePassQueryDecorator extends VehicleSearchServiceDecorator<IVehicleSearchServiceClient> {

    @Autowired
    private VehiclePassQueryFieldOptionsContext vehiclePassQueryFieldOptionsContext;

    public VehiclePassQueryDecorator(IVehicleSearchServiceClient iVehicleSearchServiceClient) {
        super(iVehicleSearchServiceClient);
    }

    /**
     * 默认分页查询
     * @param vehiclePassReqDTO
     * @return
     */
    public VehiclePassRespDTO defaultPaging(VehiclePassReqDTO vehiclePassReqDTO) {
        return customPaging(vehiclePassReqDTO, VehiclePassRespDTO.class);
    }

    /**
     * 默认查询详情
     * @param passId
     * @return
     */
    public VehiclePassRespDTO.VehiclePass defaultDetailing(String passId) {
        VehiclePassRespDTO.VehiclePass vehiclePass
                = detailByUniqueKey(passId, (vehiclePassReqDTO) -> vehiclePassReqDTO.setPassId(passId));
        return vehiclePass;
    }


    /**
     * 根据sequenceId查询详情
     * @param sequenceId
     * @return
     */
    public VehiclePassRespDTO.VehiclePass queryDetailingBySequenceId(String sequenceId) {
        VehiclePassRespDTO.VehiclePass vehiclePass
                = detailByUniqueKey(sequenceId, (vehiclePassReqDTO) -> vehiclePassReqDTO.setSequenceId(sequenceId));
        return vehiclePass;
    }


    /**
     * 自定义查询
     * @param vehiclePassReqDTO 自定义子类
     * @param rClass 返回对象
     * @return
     */
    public <P extends VehiclePassReqDTO, R> R customPaging(P vehiclePassReqDTO, Class<R> rClass) {
        if (ObjectUtil.isNull(vehiclePassReqDTO.getPermission()) || !vehiclePassReqDTO.getPermission()) {
            return (R) new VehiclePassRespDTO(0, 1);
        }
        QueryFieldOptions queryFieldOptions = vehiclePassQueryFieldOptionsContext.executeStrategy(vehiclePassReqDTO);
        BaseRspDTO<LinkedHashMap> baseRspDTO = super.query(queryFieldOptions);
        R vehiclePassRespDTO = Optional.ofNullable(baseRspDTO).map(BaseRspDTO::getBigData)
                .map(m -> BeanUtil.mapToBean(m, rClass, false)).orElse(null);
        return vehiclePassRespDTO;
    }

    /**
     * 根据唯一键查询详情
     * @param uniqueKey
     * @param setKeyFun
     * @return
     */
    public VehiclePassRespDTO.VehiclePass detailByUniqueKey(String uniqueKey, Consumer<VehiclePassReqDTO> setKeyFun) {
        if (StrUtil.isBlank(uniqueKey)) {
            return new VehiclePassRespDTO.VehiclePass();
        }
        VehiclePassReqDTO vehiclePassReqDTO = new VehiclePassReqDTO();
        vehiclePassReqDTO.setPermission(true);
        setKeyFun.accept(vehiclePassReqDTO);
        VehiclePassRespDTO vehiclePassRespDTO = defaultPaging(vehiclePassReqDTO);
        VehiclePassRespDTO.VehiclePass vehiclePass = Optional.ofNullable(vehiclePassRespDTO)
                .map(VehiclePassRespDTO::getMatchList)
                .filter(CollUtil::isNotEmpty)
                .map(l -> l.get(0)).orElse(new VehiclePassRespDTO.VehiclePass());
        return vehiclePass;
    }
}

创新点及效果 ​

​ 本案例通过整理归纳xfusion查询接口特征以及分析现有代码优劣，总结实现一套基于策略模式、处理器模式以及装饰者模式的条件组装查询方案，创新点及效果大致如下：

1、首先是代码量肉眼可见的缩减，原本大段的重复逻辑转变为添加注解即可实现条件注入；

2、代码扩展性变得更好，扩展一个条件即扩展一种策略或处理器，并且可以得到很好的复用，由之前的“纵向扩展”变成“横向扩展”；

3、代码拥有更好的可维护性，假设xfusion升级之后某条件策略关键字或语法修改，直接修改对应策略即可实现平滑升级；

4、统一条件处理入口，封装核心条件组装逻辑，提高代码健壮性同时降低了调用者的使用成本（如果对内部逻辑不求甚解，其实只需要加一个注解即可实现莫条件注入）；

5、对于“描述性”条件参数类的接口（好像大数据对外接口都是此规范）可以参考复用，该方案具备一定的可复用性。
