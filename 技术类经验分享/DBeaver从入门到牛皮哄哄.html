DBeaver从入门到牛皮哄哄 ​

入门 ​

简介 ​

​ DBeaver 是一个基于 Java 开发，免费开源的通用数据库管理和开发工具。采用 Eclipse 框架开发，支持插件扩展，并且提供了许多数据库管理工具：ER 图、数据导入/导出、数据库比较、模拟数据生成等。

​ DBeaver 通过 JDBC 连接到数据库，可以支持几乎所有的数据库产品，包括：MySQL、PostgreSQL、MariaDB、SQLite、Oracle、Db2、SQL Server、Sybase、MS Access、Teradata、Firebird、Derby 等等。商业版本更是可以支持各种 NoSQL 和大数据平台：MongoDB、InfluxDB、Apache Cassandra、Redis、Apache Hive 等。

​ 反正说来说去就是很好用、很厉害、很牛皮，在免费的圈子里基本好评如潮。

常用快捷键 ​

📌 快捷键冲突怎么办？

在Win10操作系统中，Ctrl + Shift 相关快捷键很容易和输入法切换快捷键冲突，这时候可以考虑修改输入法或者DBeaver的快捷键（建议修改输入法快捷键，关闭Ctrl + Shift + F和Ctrl + 空格即可）。

  • Ctrl + N 新建（项目、文件、数据库连接）
  • Ctrl + ] 新增一个sql执行窗口
  • Ctrl + Enter 执行sql
  • Ctrl + \ 执行sql,保留之前窗口结果
  • Ctrl + Shift + ↑ 向上复制一行
  • Ctrl + Shift + ↓ 向下复制一行
  • Ctrl + Shift + F 对sql语句进行格式化，对于很长的sql语句很有用
  • Ctrl + D 删除当前行
  • Ctrl + Alt + 空格 sql模板
  • + ↑ 向上选定一条sql语句
  • + ↓ 向下选定一条sql语句
  • Ctrl + / 行注释
  • Ctrl + Shift + / 块注释
  • Ctrl + 空格 sql提示

温馨提示 ​

  • 建立数据库连接时候，勾选“显示非缺省的数据库”就会显示当前连接下所有的数据库实例；

  • 执行查询时候，只需要将光标移动到某一条sql语句上就行，不需要选定整条完整的语句；

  • 在sql查询界面，按住Ctrl 键点击对应的表名，可以直接跳转到对应的表编辑界面；

  • 在数据查询框内编辑操作数据或者数据数据，记得点击保存按钮，不然修改或删除无效；

  • 涉及数据更新以及数据删除的sql语句操作，可以设置手动提交事务，防止误操作；

  • 部分快捷键如果因为冲突或者用的不是特别顺手，建议自定义一波；
